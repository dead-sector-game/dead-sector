# Dead Sector

> A cyberpunk roguelike arcade shooter where every run is different.

![License](https://img.shields.io/badge/license-TBD-blue)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey)
![Status](https://img.shields.io/badge/status-active-success)

Dead Sector is a fast-paced cyberpunk roguelike arcade shooter set inside hostile cyberspace. Battle through procedurally generated sectors, defeat increasingly dangerous ICE, and build unique hacking loadouts every run.

---

## Features

- Fast-paced arcade combat
- Randomized upgrades and builds
- Cyberpunk aesthetic
- Branching progression
- Increasing difficulty across multiple sectors
- Highly replayable roguelike gameplay

---

## Getting Started

### Clone the repository

```bash
git clone https://github.com/dead-sector-game/dead-sector.git
cd dead-sector
```

### Run

> Replace these instructions with your engine/framework specific setup.

```bash
bun i
bun dev
```

or

```bash
npm i
npm dev
```

or

```bash
pnpm i
pnpm dev
```
